Dubai Real Estate




The real estate agency website is targeted at businesses, potential investors and prospective buyers or tenants in the UAE. The application allows users to search for all available properties using a wide range of search filters to cater to their needs. The website includes a homepage, followed by two pages, one for rental and one for purchasing a property. The application uses React, specifically the NextJS framework, ChakraUI for styles, and NodeJS. All the data is coming from an API, and any generated data can be updated on the server using NextJS. The hamburger menu in the top right corner allows for easy navigation of the website.
